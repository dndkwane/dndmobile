우리은행
오픈뱅킹 시스템

우리은행의 브랜드 색상을 적용하고 고해상도의 깔끔한 이미지와 아이콘을 배치해
모던한 분위기의 디자인을 적용했습니다.

Design Concept

우리은행에 위비 뱅크 앱에 복잡한 기능을 사용자가 편리하게 사용할 수 있도록 재구성하고
브랜드 아이덴티티가 일관성있게 적용될 수 있도록 디자인시스템을 정리하고 UI를 직관적으로 설계하였습니다.
