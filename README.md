## Getting Started

### Installing

* How to run the program
* Step-by-step bullets

1. Clone the repo
```
git clone https://github.com/misunyang/dndmobile.git
```
2. Install NPM packages
```
npm install
```
```
npm install -g gulp
```
```
npm install --save-dev gulp
```
3. Local start
```
gulp
```
